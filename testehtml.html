Questionário
Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

Resposta: Servidor WEB entrega os arquivos que já estão prontos, como html, css e imagens, um dos seus exemplos é o servidor do Apache.
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta: Ele é um software de código aberto que possui proxy reverso, manuseio eficiente de arquivos estáticos e de index, balanceamento de carga, descarregamento de SSL/TLS, além de armazenamento em cache e compactação.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

Resposta:Os dois são protocolos de transferência de dados na internet, porém em HTTPS os dados são criptografados, garantindo mais segurança para a aplicação.
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

Resposta:Portas estabelecem as conexões entre aplicações e usuários, as portas 443 são específicas para conexões de HTTPS.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

Resposta:Trata-se da identificação de um dispositivo na internet, servindo também como endereço virtual do dispositivo, importante para identificar os dispositivos e enviar adequadamente os dados aos respectivos solicitantes.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

Resposta: Trata-se de um serviço de hospedagem de um servidor físico compartilhado, permitindo maior controle e desempenho para a hospedagem e custando menos que um servidor dedicado.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

Resposta: O Caddy possui HTTPS automático, até para localhost, é mais simples de configurar, além de sua arquitetura moderna que garante desempenho leve e eficiente.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

Resposta: O ASP.NET Core é o mais indicado para esse tipo de aplicação, devido a sua estrutura multiplataforma e software livre, o que garante desenvolvimento de aplicações tanto para Windows, Linux os MacOS. Além de ser indicado para desenvolvimento de aplicativos em grande escala e grandes cargas de trabalho, tornando-se uma escolha principal para empresas.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

Resposta: O cliente faz somente os pedidos ao sistema e o servidor responde e entrega de acordo com aquilo que foi requisitado.
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

Resposta: É um servidor web de código aberto e contêiner de servlet desenvolvido pela Apache Software, permite que você execute aplicações web que foram escritas em Java.
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

Resposta: Servidor web entrega somente aquilo que já está pronto e não realiza processamento de códigos, gerando dados estáticos, diferente dos servidores de aplicação que tratam, principalmente, da lógica de programação nos servidores, trazendo resultados mais dinâmicos.
Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

Resposta: É o nome de um IP, uma forma mais simplificada de buscar e identificar um endereço na internet, como o site do google, que possui o domínio www.google.com.
Questão 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

Resposta: Baixo custo para manutenção e implementação, além de não exigir um espaço físico para que possa funcionar.
Questão 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

Resposta: Node.Js que é usado principalmente para aplicações backend em JavaScript e Django, um framework de Python também usado principalmente para o mesmo propósito.
Questão 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

Resposta: O HTTP é considerado um protocolo de transferência de dados inseguro, pois não garante a criptografia dos dados transferidos, permitindo que atacantes possam interceptar a rede e olhar as informações, o que não ocorre em HTTPS.
Questão 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

Resposta: Para aplicações que tenham alta demanda e que precisem de mais flexibilidade, pois apesar de ser mais cara, oferece mais recursos dedicados e escalabilidade para que possa lidar com pico de tráfego em momentos críticos.
Questão 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

Resposta: IP é o endereço do dispositivo enquanto a porta é o canal de comunicação onde os dados são transferidos.
Questão 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

Resposta: Garantir a segurança dos dados das aplicações.
Questão 19: Cite dois exemplos de servidores web e explique suas principais características.

Resposta: Microsoft IIS, possui integridade nativa com Windows Server. LiteSpeed, servidor de alta performance e compatível com Apache.
Questão 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

Resposta: Hospedagem compartilhada é boa para projetos iniciantes e com baixo tráfego, pois possui baixo custo e possui recursos suficientes para atender a demanda.
Questão 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

Resposta: Servidores de aplicações entregam respostas dinâmicas e fazem tratamento de dados aplicados.
Questão 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

Resposta: Gunicorn, ele atua como intermediário entre um servidor web e a aplicação python, exigindo participação de outro servidor para requisições HTTP.
Questão 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

Resposta: Servidor de aplicações para Java, utilizado para aplicações web, microsserviços e sistemas de mensagem.
Questão 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

Resposta: A porta 21 é usada principalmente para conexões de controle no protocolo FTP (File Transfer Protocol).
Questão 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

Resposta: Facilita a migração de sites hospedados em Apache para o LiteSpeed, pois não é necessário reconfigurar o servidor ou ajustar as aplicações web.
Questão 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

Resposta: Trata-se de um servidor web da Microsoft, costuma ser adotado em casos quando você precisa de um servidor web para hospedar sites, aplicativos e serviços web, especialmente em ambientes Windows.
Questão 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

Resposta: Ele é indicado para aplicações backend que utilizam a linguagem Javascript.
Questão 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

Resposta: É um servidor de aplicações de código aberto que executa aplicações Java.
Questão 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

Resposta: Puma e Unicorn são servidores para aplicações Ruby on Rails que lidam com requisições HTTP.
Questão 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

Resposta: É importante para que se utilize aquele que mais atenda ao requisito da aplicação e que o custo para implementação e execução não seja alto.
